# IBM-Prompt-Engineering-Lab
A hands-on lab exploring advanced prompt engineering techniques using IBM Granite 3.3 models via Replicate and LangChain in Google Colab
Objective: To maaster zero-shot and structured prompting for professional reporting
Tools Used: Google Colab, Replicate API, and IBM Granite 3.3-8b-instruct model.
Key Learning: Transitioning from simple prompts to structured frameworks(Role, Task, Tone, Format) to improve AI output accuracy.
Mastered Variable Templates: Built reuseable prompt frameworks for consistent corporate reporting at scale.
